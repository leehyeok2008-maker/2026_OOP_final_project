# 2026_OOP_final_project
2026학년도 객제지향프로그래밍(Object-Oriented Programming) 수업 최종 프로젝트 레포지토리

# 코드 규칙
1. 지역 변수: 소문자, 명사
e.g. position = (1, 1)

2. 전역 변수: 대문자, 명사
e.g. WIDTH = 720 

3. 함수 및 메서드 : 소문자, 동사 
e.g. def change_scene(scene_name : str)

4. 클래스: 첫 글자 대문자, 명사
e.g. class HeavyCargo(Entity)

5. 파일: 소문자, 명사
e.g. stage.py
